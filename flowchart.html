Flow 14 — Pricing & Commission System 🆕
💰 Pricing Engine
Three-layer pricing chain — purchase + margin + commission = final price
Purchase Price (₹) + Branch Profit Margin (₹) + Staff Commission (₹) = Final Sale Price (₹)
Set by Staff / Manager  ·  Set by Branch Manager  ·  Optional, set by Sales Staff  ·  Auto-calculated
💵 Purchase Price
Who Sets This
  • Sales Staff OR Branch Manager
  • Per unit / per model
  • Reflects actual cost paid
  • Set at time of inventory entry
📊 Branch Profit Margin
Branch Manager Only
  • Set per device model, per branch
  • Fixed amount in ₹
  • Different branches can have different margins
  • Effective from date tracked
  • Margin history stored
  • ⚠️ Changes don't affect past sales
💸 Staff Commission
Sales Staff (Optional)
  • Per individual sale transaction
  • Added at time of sale
  • Can be ₹0 (no commission)
  • Tracked separately from profit
💡 Example: Samsung F22 purchase = ₹20,000. Branch margin = ₹4,000. Base price = ₹24,000. Staff commission = ₹500. Final sale price = ₹24,500. Branch profit = ₹4,000. Staff commission = ₹500 (tracked separately).
⚠️ Critical Rule: Changing the branch profit margin does NOT affect previously recorded sales. The margin at the time of each sale is saved permanently with that transaction.
