Flow 2 — Super Admin: Dashboard & Controls
👑 Super Admin Dashboard
Platform-level owner — manages all tenants
Super Admin sees sidebar/menu with these options ↓

Overview Panel

  • Active users per tenant
  • Tenant account status (Active / Suspended)
  • Casual summary view
🏢 Tenant Management
Manage Tenants & Users
  • Approve / reject new tenant registrations
  • Set max user accounts per tenant
  • Suspend or reactivate tenant
  • Reset tenant user passwords
🎛️ Feature Toggles
Per-Tenant Feature Controls
  • Enable / disable features per tenant
  • Hiding a feature removes the UI button
  • No data is deleted when toggled off
  • See toggle table below ↓

Flow 4 — Service Detail Page: What User Can Do Here
📋 Service Detail / Update Page
Reached from: Add Service OR clicking a job in Service List
This page shows job card info and offers these actions ↓
👁️ View Info
Job Card Details
  • Serial Number
  • Customer Name & Phone
  • Brand / Model / IMEI
  • Problem Description
  • Received Date
  • Assigned Technician
  • Current Status
  • Service History log
🔄 Update Status
Change Status
  • Select new status:
  •   Received → Checking
  •   Checking → Waiting
  •   Waiting → Repaired
  •   Repaired → Delivered
  •   Any → Cancelled
  • Click "Update" → saved
  • → Logged in service history
🔩 Add Parts Used
Parts Source Selection
  • Option A: Shop Part
  •   Search / select from catalogue
  •   Auto-fill cost & selling price
  •   ⚠️ Cost locked at this time
  • Option B: Outside Purchase
  •   Select "Purchased from Outside"
  •   Enter part name manually
  •   Enter cost of the product
  • Enter quantity
  • Click "Add" → attached to job
  • Can add multiple parts
📝 Add Notes
Notes Section
  • Technician repair notes
  • What was fixed/repaired
  • Customer approval notes
  • Update delivery date
🖨️ Print
Print Options
  • Print Job Card
  • Print Service Receipt
If enabled by Super Admin

User Roles
👑 Super Admin
Platform owner — full control over tenants, feature toggles, user quotas, account approvals
🏪 Tenant Admin (Main Branch)
Shop owner/manager — create jobs, manage parts & models, own shop data only. Can manage sub-branches.
🏢 Sub-Branch User
Branch staff — restricted to viewing and managing jobs only for their specific branch. Cannot manage other branches.
🛠️ Technician User
Technician staff — restricted strictly to viewing their assigned jobs and updating status/notes. Cannot create jobs.
